GrIP is a Mathematica® based package that allows the construction of Group Invariant Polynomial for any given model. The user needs to provide GrIP with an input model file that contains the information about symmetry groups, (super)field content and the transformation properties of the (super)fields under given groups. GrIP uses this input file to form the Hilbert Series that consists of possible operator structures. Set of operators for certain mass/canonical dimension(s) can be filtered out using proper functions.
